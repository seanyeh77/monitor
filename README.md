# Setup Instructions

## Step 1: Install CUDA Toolkit

Install the CUDA Toolkit from NVIDIA.

Reference: https://developer.nvidia.com/cuda-downloads

## Step 2: Install CUDA Container Toolkit

Install the CUDA Container Toolkit to enable GPU support in Docker containers.

Reference: https://docs.nvidia.com/datacenter/cloud-native/container-toolkit/install-guide.html

## Step 3: Start the Application

```bash
docker compose up -d
```
